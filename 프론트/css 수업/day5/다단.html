선택자 : 요소들을 선택할 수 있게 하는 것 (어떻게 시작하는지 정도는 외워야 함 *, #, . , ...)

1) 태그 선택자 : 태그 이름을 사용, 해당 태그 전체에 적용 2) 클래스 선택자 . 태그의 class 속성의 이름을 이용하는 방법으로 클래스명 앞에 . 을 붙임 . 해당 클래스를 가지고 있는 요소들을 선택 . 클래스는 여러개를 띄워쓰기로 추가할 수 있음 3) 아이디 선택자 . 태그의 id 속성을 지정하여 id를 이용하는 방법 . 아이디명 앞에 #을 붙임 . 아이디는 중복되면 안됨 4) 가상 클래스 선택자 . 특정 상황에 동작하는 선택자 . 종류 많음 - hover:마우스를 요소 위에 올렸을때 - active:마우스로 요소를 클릭 중 일때 - focus:요소가 포커스 받고 있을 때(input태그에 커서가 활성화 되었을 때) - visited:방문한 링크일 때(a태그)) - nth-child(숫자)/nth-of-type(숫자) : 해당 선택자와 일치하는 숫자번째 요소를 선택 * 숫자인 경우 해당 숫자번째 요소 1개 선택 - nth-child(2n)/nth-of-type(2n) : 해당 선택자 중에서 짝수번째 요소들을 선택 - nth-child(even)/nth-of-type(even) : 해당 선택자 중에서 짝수번째 요소들을 선택 - nth-child(2n-1)/nth-of-type(2n-1) : 해당 선택자 중에서 홀수번째 요소들을 선택 - nth-child(odd)/nth-of-type(odd) : 해당 선택자 중에서 홀수번째 요소들을 선택 -> 태그에 따라 nth-child가 적용되는 태그가 있고 nth-of-type이 적용되는 태그가 있음 - ::before -> 요소 앞에 새로운 요소를 추가하는 선택자 -> 요소 앞에 가운데 점 또는 | 또는 아이콘과 같은 요소를 추가할 때 사용 - ::after -> 요소 뒤에 새로운 요소를 추가하는 선택자 5) 전체 선택자 : *로 모든 요소에 적용 6) 속성 선택자 : 속성을 이용하여 요소를 선택 . 선택자[속성] : 속성이 포함된 요소들을 선택 . 선택자[속성=값1] : 속성의 값이 값1과 같은 요소들을 선택 * 가장 많이 쓰임 . 선택자[속성^=값1] : 속성의 값이 값1로 시작하는 요소를 선택 . 선택자[속성$=값1] : 속성의 값이 값1로 끝나는 요소를 선택 . 선택자[속성~=값1] : 속성의 값이 값1을 포함하는 요소들을 선택(공백을 기준으로 분리된 값이 포함) [ class~="col"] : class는 "col row" 인 경우는 포함됨 class="colrow"인 경우는 포함이 안됨 . 선택자[속성*=값1] : 속성의 값이 값1을 포함하는 요소들을 선택 [ class*="col"] : class는 "col row" 인 경우는 포함됨 class="colrow"인 경우도 포함됨 . 선택자[속성|=값1] : 속성의 값이 값1이거나 값1로 시작하는 요소들을 선택 7) 선택자를 이용한 css 적용 방법(방법2 또는 방법3) . 기본문법 선택자 { 속성1 : 값1; 속성2 : 값2; ... ;
